css margin视屏教程

margin重叠通常特性

1. block水平元素(不包括float和absolute)

2.不考了writin-mode,只发生在垂直方向

margin重叠3种场景

  • 1.相邻的兄弟元素
  • 2.父级和第一个或最后一个重叠
  • 3.空的block元素

父子margin重叠其他条件

margin-top重叠

1.父元素非块状格式化上下文元素(如:1float,2border:1px solid #ff0 3overflow:hidder)

2.父元素没有border-top设置

3.父元素没有padding-top值设置

4/父元素和第一个子元素之间没有inline元素分隔(如：在父元素后面加 )

margin-bottom重叠

1.父元素非块状格式化上下文元素(如:1float,2border:1px solid #ff0 3overflow:hidder)

2.父元素没有border-bottom设置

3.父元素没有padding-bottom值设置

4.父元素和第一个子元素之间没有inline元素分隔

5.父元素没有height,min-heigh,max-height

重叠的计算规则

1.正正取大值;

2.正负值相加;

3.负负最负值;
